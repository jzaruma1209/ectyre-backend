# Workflows

Aquí irán los workflows de Antigravity.

Un workflow es una secuencia de pasos que el agente debe seguir para completar una tarea compleja,
por ejemplo: "onboarding de nuevo proyecto Node.js" o "deploy a producción en Vercel".

## Próximos workflows

- [ ] `nuevo-proyecto-node` — Setup completo de un proyecto Node + Sequelize desde cero
- [ ] `deploy-vercel` — Pasos para hacer deploy en Vercel correctamente
- [ ] `nuevo-modulo-api` — Checklist completo para agregar un módulo a la API
